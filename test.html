Lettering test drive

All succeeded letterings should be green, otherwise they are red.

Using ID's

<div id="example-1">Text</div>
lettering( document.getElementById("example-1") );

Result

text

Using classes

<div class="clazz">clazzified</div>
<div class="clazz">clazzified as well</div>
lettering( document.getElementsByClassName("clazz") );

Result

clazzified
clazzified as well

Using elements

<article>Dah Article</article>
<article>Dah Article also isse cool!</article>
lettering( document.getElementsByTagName("article") );

Result

Dah Article
Dah Article also isse cool!

Including line–breaks

Lettering can play nicely with the break tag.

<div id="example-4">Text<br/>and more</div>
lettering( document.getElementById("example-4") );

Result

Text
and more

Including inline tags

Lettering can recognize inline tags one level deep.

<div id="example-5">Text <em>and<em> <strong>more<strong></div>
lettering( document.getElementById("example-5") );

Result

Text and more link

Using the span element

Lettering tries not to break when already using spans

<div id="example-6">Text <span>here<span></div>
lettering( document.getElementById("example-6") );

Result

Text here

Breaking the function

Here we are looking for a non-existing element with id 'non-existing-id'. Remember to put all calls to lettering inside a try-catch block, to prevent breaking the complete program.

<aside>Derp</aside>
lettering( document.getElementById("non-existing-id") );

Result

Derp
